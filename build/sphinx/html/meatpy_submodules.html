class meatpy.market_processor.MarketProcessor(instrument, book_date)¶

Bases: object

A market engine that process events and builds the limit order book

A MarketProcessor object is an engine that processes MarketMessage objects to build the market history for one instrument, on one day and on one market.

The MarketProcessor keeps an up-to-date version of the limit order book, unless instructed not to.

class meatpy.timestamp.Timestamp¶

Bases: datetime.datetime

Basic timestamp derived form datetime

The timestamp is derived from the standard datetime class. The main is that precision is limited to microseconds. Timestamps need to be comparable (<, <=, ==, !=, >=, >) and need to have a __str__() method.
