meatpy.itch50 package¶

Package contents¶

This Subpackage is for tracking and recording events that happened in ITCH, version 5.0.

About¶

This package adds support for the Nasdaq TotalView-ITCH 5.0 file format. For more details, see the format specification.

Known issues¶

Two message types are not supported yet:

  • LULD Auction collars message (introduced September 6, 2017)

  • Operational Haltmessage (introduced March 3, 2018)

Daily files after those dates might not process properly. We are planning to add support for these messages soon.

meatpy.itch50.itch50_ofi_recorder module¶

A recorder for the changes in OFI adapted to ITCH 4.1

See Equations (4) and (10) of Cont, R., et al. (2013). “The Price Impact of Order Book Events.” Journal of Financial Econometrics 12(1): 47-88.

The recorder follows equation (10) but accounts for trades against hidden orders as well.
