meatpy package¶

Package contents¶

The goal of this project is to provide a standard framework for processing and analysing high-frequency financial data. MeatPy can be used to process event-time data from multiple financial exchanges and formats, such as Nasdaq ITCH.

This package contains some subpackages and submodules:

Subpackages:

  • event_handlers

  • itch50

Submodules:

  • level

  • lob (Limit Order Book)

  • market_event_handler

  • market_processor

  • message_parser

  • timestamp

  • trading_status
